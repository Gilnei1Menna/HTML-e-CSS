Código-fonte / Pré-formatação

Para usar letras mono-espaçadas usamos a tag CODE

A tag PRE serve para manter o texto da maneira como foi digitado

EX. de um código em python, sem a tag PRE

num = int(input('Digite um número')) if num % 2 == 0: print(f'0 número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

EX. de um código em python, com a tag PRE

        
            num = int(input('Digite um número'))
            if num % 2 == 0:  
                print(f'O número {num} é PAR')
            else: 
                print(f '0 número {num} é PAR')
            print('Fim do programa')
