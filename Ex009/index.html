Trabalhando com listas

Listas Ordenadas

No exemplo a baixo vemos a tag OL para criar uma lista ordenada automaticamente, com o auxilio do parametro TYPE= que da opção de 3 tipos (Numérica[1], Alfabética[A/a] ou Romana[I/i]) e o START que seleciona quando começa a contagem podemos dar mais detalhes a lista.

  1. Acordar
  2. Meditar
  3. Tomar banho
  4. Tomar café
  5. Escovar os dentes

Listas não Ordenadas

Vemos o mesmo exemplo só que utilizo a tag UL para criar uma lista sem ordem e usando o parametro TYPE= que aqui tem 3 opções (square, disc e circle)

  • Acordar
  • Meditar
  • Tomar banho
  • Tomar café
  • Escovar os dentes

Criando listas dentro de listas

É bem simples na verdade, é só colocar ou tag OL ou a tag UL dentro da tag LI e tambem podemos usar todos os parametros dos exemplos a cima.

EX.

  1. Acordar
    • 5:00
  2. Meditar
  3. Tomar banho...
