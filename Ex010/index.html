Usando hyperlinks

Liks externos

Para criar um hiperlink utilizamos a tag A com um atributo, aqui usamos o HREF e também o atributo TARGET com o valor _SELF para indicar que queremos permanecer na janela atual. E para ir para outra janela usamos valor _BLANK dentro do atributo TARGET.

Outro recurso que percebemos eo do atributo REL, que indica a natureza do link, no caso do valor EXTERNAL, ele indica que o link não faz parte do documento atual.
outros valores para o atributo REL são:
‣NEXT indica que o link é para a próxima parte do documento atual
‣PREV indica que o link é para a parte anterior do documento atual
‣AUHTOR indica que é um link para o site do autor do artigo atual
